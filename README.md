# Gerador de QR Code

Projeto simples de **Gerador de QR Code** desenvolvido com **HTML, CSS e JavaScript puro**.

O usuário digita um texto ou link, clica no botão para gerar e o sistema cria um QR Code automaticamente usando uma API externa.

## Preview

Este projeto permite:

- digitar um texto, link ou qualquer informação
- gerar um QR Code em tempo real
- exibir o QR Code na própria interface
- usar uma interface simples e objetiva

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- API de geração de QR Code

## Como funciona

O usuário insere um valor no campo de texto.  
Ao clicar no botão, o JavaScript captura esse valor e monta a URL da API responsável por gerar a imagem do QR Code.

Depois disso, a imagem é exibida na tela.
